# Public Scope

Use this repository only for hackathon-safe public assets.

## Safe To Publish

- Product summary
- Demo steps
- Architecture diagram at a high level
- Non-sensitive API examples
- Deployment notes without secrets
- Screenshots and pitch materials

## Do Not Publish

- Real API keys or `.env` values
- Full production adapters
- Fraud, policy, or risk internals
- Customer or partner information
- Detailed commercialization strategy
- Non-public operational runbooks

## Repository Strategy

- Keep the primary `agent-payment` repository separate.
- Use this repository as the public submission shell.
- Copy in only sanitized demo assets.
