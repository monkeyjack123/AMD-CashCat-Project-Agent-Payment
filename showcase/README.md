# Showcase Code

This folder contains selected public-safe source excerpts and pseudocode for the CashCat AI runtime.

Purpose:

- show judges and reviewers how the AI workflow is structured
- expose planner, orchestration, and governed spend concepts without publishing the full internal repository

Included:

- source excerpts from the showcase runtime
- AI runtime design notes
- planner prompt and output-shape pseudocode
- orchestration decision logic pseudocode

Notes:

- these files are excerpts, not a standalone production app
- sensitive modules and commercialization details are intentionally omitted
- the goal is to show real AI workflow design, not full private implementation details
